# 28/02/2022
 - La mise a jour automatique toutes les 5 minutes est de nouveau disponible pour tous les équipements.
   Pour éviter la facturation Google, un cache a été ajouté, ainsi qu'un nombre maximum de requête par
   jour pour le plugin. Par défaut, les informations dans le cache sont considérées comme valide si elles
   ont moins de 5 minutes, ou si l’équipement ne s'est pas déplacé (les coordonnées source et destination
   sont les memes).
